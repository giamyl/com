# bcom

写代码中用到的一些公共的函数和方法

将平时经常用到的东西记录下来， 避免用到的时候再找浪费时间

## 自定义函数包

暂时还简单， 根据实际使用慢慢完善

+ bpath 用来处理路径， 比如获取项目根目录

+ blog 用来处理日志
